# brunoconterato

Espaco de notas, notebooks e exercicios do curso.

## Guia principal para pesquisa web local

Se voce quiser entender e reproduzir o stack de pesquisa web usado em `2_openai/my_deep_research.ipynb`, leia:

- [Local Web Research Stack](guides/local-web-research-stack.md)

Esse guia explica:

- como o notebook usa Ollama local;
- como a busca web e feita por um DDGS local na porta `4479`;
- como baixar e extrair o texto das paginas;
- como rodar tudo do zero em uma maquina nova com `uv` e `docker`.

## Ordem pratica para começar

1. Leia o guia acima.
2. Suba o Ollama local.
3. Suba o DDGS local.
4. Rode os notebooks ou scripts dentro de `brunoconterato`.
